# PhysicsEngine-DrawTool
## 概要
物理エンジン(OBAQ)を使った抽象画を作るツール、誰でもポロックになれる。
## 詳細
- 物理演算(OBAQ)  
キーボードで操作するオブジェクトの軌跡を筆の軌跡とみなした。スピードに応じた円を描画した。また、モブオブジェクトに当たった場所にも円を描画した。
- 喜怒哀楽パレット  
喜⇔哀、怒⇔楽と考えた。喜は黄色、怒は赤、哀は青、楽は緑とし、表現したい感情に合った色で描画できるようにした。
